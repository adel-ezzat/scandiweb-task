## Installation

1. clone the repo
2. `cd projectname/fronted` run `npm install && npm run build` .
3. import database `scandiweb.sql` .
4. configure mysql in `Config/Database.php` .

## Requirements

1. PHP.
2. MySQL.
3. nodejs.
